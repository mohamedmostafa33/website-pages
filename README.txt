Description:
This project is a simple website pages consisting of three HTML files and one CSS file for styling. It is designed to be viewed in any modern web browser.

Files Included:
index.html
about.html 
projects.html
styles.css

How to View the Website:
Step 1: Download all the files (HTML and CSS) to a local folder on your computer.
Step 2: Open the index.html file in any web browser (Google Chrome, Firefox, Microsoft Edge, etc.) by double-clicking on the file or right-clicking and selecting "Open with" and choosing your browser.
Step 3: Navigate between pages using the links provided in the website's navigation menu.

Notes:
Ensure that the styles.css file is located in the same directory as the HTML files for the styles to load correctly.
